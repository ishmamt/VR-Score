# VR-Score
VR-Score: A Multi-Modal Metric for Quantifying AI Video Realism

## Getting Started
- It is better to run this project using `uv` for managing and installing dependencies. Learn more about uv from their <a href="https://docs.astral.sh/uv/getting-started/">official docs</a>. To install `uv` through `pip`, you can use:
```
pip install uv
```

- Clone the GitHub repository"
```
git clone https://github.com/ishmamt/VR-Score.git
```

- Change into the directory and initialize `uv`:
```
cd VR-Score
uv init
```

- To install all the dependencies:
```
uv sync
```

- To run the project:
```
uv run main.py
```
